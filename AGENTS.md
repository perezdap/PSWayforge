# PSWayforge

Agent instructions for **PSWayforge** — a PowerShell module that scaffolds
agent-agnostic project workspaces.

## Agent skills

### Issue tracker

Issues live as GitHub issues in `perezdap/PSWayforge`, managed with the `gh` CLI.
See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage roles, each label string equal to its name.
See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — `CONTEXT.md` and `docs/adr/` at the repo root.
See `docs/agents/domain.md`.
